> 增强兔兔的查询功能，让他可以执行更多操作，相比于原来的兔兔查询，现在他可以

    - 在查询关卡掉落时,显示材料掉率数字
    - 在使用公招查询时，可以不使用百度Api，而是进行本地图片识别查询。（需要安装PaddleOcr）

> 关于PaddleOcr

    - 如果想要使用本地Ocr，请在命令行执行 `pip install paddleocr` 和 `pip install paddlepaddle` 来安装
    - 可执行文件部署方式，暂时无法支持该功能。
